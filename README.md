# 1. Introduction

Hello, This project contains an analysis of INC 5000 Companies using SQL queries. The dataset contans information about each company on the INC 5000 list in 2019. Fields include the company name, industry, founding year, website, and location, as well as 2019 revenue, % growth, number of workers (year-over-year), and the number of years on the list.

# 2. Objectives- To show;
   - Average revenue among companies on the list? Broken down by industry?
   - Industries that are most represented and least represented.
   - Geographic trends among the companies
   - Industries that saw the largest  average growth rate
   - If any company was able to increase revenue while reducing staff
   - Companies that had the largest increase in staff/new hires
   - Industries average staff turnover
   - performance metrics across companies in different industries
